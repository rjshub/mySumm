1、让href的值为javascript:;

点击不会跳转
<a href="javascript:;">点击不会跳转</a>

2、让点击事件直接返回false

点击不会跳转
<a href="#" onclick="return false">点击不会跳转</a>

3、通过js阻止默认行为

点击不跳转
<a href="#" id="test">点击不跳转</a> <script> document.getElementById("test").onclick = function (e) { e.preventDefault() } </script>

4、通过css让元素不响应任何鼠标事件

(设置元素是否对指针事件做出反应，none代表不做反应，auto默认值代表做出反应)
点击不会跳转
<a href="#" style="pointer-events: none">点击不会跳转</a>
