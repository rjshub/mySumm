Vue项目实现语言国际化的相关配置

1、安装vue-i18n依赖包

npm install vue-i18n -s

2、创建文件夹放置翻译文件

在src目录下（方便整理，最好与main.js同级）创建VueI18n文件夹，在该目录下创建需要翻译语种的JS文件

3、编辑创建好的语种JS文件

在相对应的JS文件中引入项目所使用的UI组件语言文件，实现在进行语言切换时UI组件不受影响，此案例为Element_UI。
语言文件创建相对应的变量，value为对应的语言文本。

4、配置main.js文件

在main.js文件中引用vue-i18n依赖包及配置好的语JS文件
// 国际化配置
import store from './store'
import Vue from 'vue'
import ElementUI from 'element-ui'
import VueI18n from 'vue-i18n'
import messages from './VueI18n'

Vue.use(VueI18n)
const i18n = new VueI18n({
locale: store.state.user.language, // 语言标识 //this.$i18n.locale // 通过切换locale的值来实现语言切换
messages
})
Vue.use(ElementUI, {
i18n: (key, value) => i18n.t(key, value)
})

new Vue({
el: '#app',
i18n,
})

5、项目中切换语言的DOM

页面实现切换语言的DOM元素，本案例为操作时更新store仓库里面定义的language变量，达到全局统一。

6、项目中全局使用语言文本

HTML代码中：$t('lang.yi_jian_zhong_qing')
JS代码中：this.$t('lang.yi_jian_zhong_qing')
